# EnabledIfInsertion Property
Sets a value that indicates whether the action should be enabled or disabled if the record is undefined.  
  
## Applies To  
  
-   Actions  
  
-   Pages  
  
## Property Value  
 **True**/**false**  

## Syntax
```
EnabledIfInsertion = true;
```
