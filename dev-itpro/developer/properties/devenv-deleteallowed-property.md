# DeleteAllowed Property
Sets a value that specifies whether users can delete records while using the page.  
  
## Applies To  
  
-   Pages  
  
## Property Value  
 **True** if the user can delete records; otherwise, **false**. The default value is **true**.  

## Syntax
```
DeleteAllowed = false;
```
