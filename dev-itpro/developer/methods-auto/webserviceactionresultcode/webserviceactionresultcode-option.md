# WebServiceActionResultCode Option Type
Represents a web service action status code.

## Members
|  Member  |  Description  |
|----------------|---------------|
|None|No status code.|
|Get|Item read.|
|Created|Item created.|
|Updated|Item updated.|
|Deleted|Item deleted.|
