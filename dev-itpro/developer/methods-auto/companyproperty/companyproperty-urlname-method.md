# UrlName Method
Gets the string that represents the company name in a URL.


## Syntax
```
UrlName :=   CompanyProperty.UrlName()
```


## Return Value
*UrlName*  
&emsp;Type: [String](../string/string-data-type.md)  
The company name in a URL.
