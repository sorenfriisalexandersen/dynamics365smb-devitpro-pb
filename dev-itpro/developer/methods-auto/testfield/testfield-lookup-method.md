# Lookup Method
Provides a lookup window for a text box on a test page.


## Syntax
```
 TestField.Lookup()
```

## Parameters
*TestField*  
&emsp;Type: [TestField](testfield-data-type.md)  
An instance of the [TestField](testfield-data-type.md) data type.
