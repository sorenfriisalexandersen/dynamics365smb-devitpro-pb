# FromInteger Method
Returns an enum with the integer value


## Syntax
```
Enum with integer value :=   Enum.FromInteger(Value: Integer)
```
## Parameters
*Value*  
&emsp;Type: [Integer](../integer/integer-data-type.md)  
  


## Return Value
*Enum with integer value*  
&emsp;Type: [Any](../any/any-data-type.md)
