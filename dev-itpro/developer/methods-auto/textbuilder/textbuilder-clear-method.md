# Clear Method
Removes all characters from the current TextBuilder instance.


## Syntax
```
 TextBuilder.Clear()
```

## Parameters
*TextBuilder*  
&emsp;Type: [TextBuilder](textbuilder-data-type.md)  
An instance of the [TextBuilder](textbuilder-data-type.md) data type.
