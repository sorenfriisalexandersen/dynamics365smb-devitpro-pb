# GetArchiveVersion Method
Returns the version of the extension that the specified table is part of.


## Syntax
```
Version :=   NavApp.GetArchiveVersion()
```


## Return Value
*Version*  
&emsp;Type: [String](../string/string-data-type.md)  
The version
