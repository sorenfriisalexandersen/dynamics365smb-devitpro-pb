# CanCreateTask Method
Checks whether it is possible to schedule tasks in this session.


## Syntax
```
Ok :=   TaskScheduler.CanCreateTask()
```


## Return Value
*Ok*  
&emsp;Type: [Boolean](../boolean/boolean-data-type.md)
