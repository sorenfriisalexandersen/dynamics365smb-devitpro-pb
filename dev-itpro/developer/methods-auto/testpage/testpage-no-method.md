# No Method
Gets the No system action.


## Syntax
```
Action :=   TestPage.No()
```

## Parameters
*TestPage*  
&emsp;Type: [TestPage](testpage-data-type.md)  
An instance of the [TestPage](testpage-data-type.md) data type.  

## Return Value
*Action*  
&emsp;Type: [TestAction](../testaction/testaction-data-type.md)  
The No system action.
